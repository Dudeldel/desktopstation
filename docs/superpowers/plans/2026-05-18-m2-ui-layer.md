This plan operationalizes M2 from `docs/plan/00-roadmap.md` (lines 69–99). M2 delivers the visual shell: top bar, 4-screen carousel with swipe + autoscroll, mock data, dark theme. Real API integrations are deferred to M3–M5.

## Design decisions (resolved without user input)

Each entry resolves an open M2 question to a "recommended" answer with rationale and a pointer to where it shows up in the plan. The user can flip any of these after reviewing.

1. **LVGL render mode → direct mode against the RGB panel framebuffer.**
   We currently use partial 800×60 buffers + `esp_lcd_panel_draw_bitmap` in `flush_cb`. This caused the residual white-bar bug we abandoned at end of M1 (`fix(firmware): panel timings`, latest cleanup commit). Direct mode hands LVGL the panel's own framebuffer; flush is just a buffer swap. Eliminates partial-flush tear, matches the spec rule "host pushes full snapshots / full redraw at every message" — same idea.
   Implemented in **Task B1**.

2. **Font → Inter, subset to Latin Extended-A (Polish glyphs), sizes 14 / 16 / 20 / 28.**
   Inter is OFL-licensed, ubiquitous, optimized for small UI. We generate `lv_font_t` C arrays via the LVGL Font Converter (online tool). For tonight we **stub the font conversion** — the firmware will reference `&lv_font_montserrat_16` etc. (LVGL built-in, no Polish glyphs but compiles). User generates real Inter fonts when convenient; one find/replace migrates.
   Implemented in **Task B2** (stub), with a TODO for user.

3. **Theme constants → match spec exactly.** `bg #0d0d10`, `card #16161a`, `accent #1d9e75`, `text_primary #f0f0f0`, `text_secondary #a0a0a0`. Plus a danger color `#d9534f` for "Disconnected" toast etc.
   Implemented in **Task B3**.

4. **Carousel → `lv_tileview` with 4 tiles, mandatory horizontal scroll snap, vertical scroll disabled.**
   Native LVGL widget, gestures supported out of the box.
   Implemented in **Task B5**.

5. **Top bar height → 40 px** (per spec). Carousel area is `480 - 40 - dot_indicator_height = 432 px`.

6. **Dot indicator → 4 dots between top bar and carousel, 6 px diameter, accent color when active, `#3a3a40` (dim) otherwise.**
   Implemented in **Task B6**.

7. **Autoscroll → 10 s base interval; touch event resets to 30 s for one cycle then back to 10 s; pomodoro fullscreen pauses.**
   Implemented as `lv_timer` in **Task B7**.

8. **Screen state → ephemeral on firmware.** No persistence across reboot. Per the spec's three rules, host re-pushes full snapshots on reconnect.

9. **Mock data → one async task per screen + top bar in `deskstation/pollers/mock.py`.** Configurable interval per screen (default 5 s). Generates plausible Polish-language content (PR titles, ticket summaries, todo items, weather, etc.).
   Implemented in **Task A4**.

10. **ui_state aggregator → singleton-ish class `UIState` in `deskstation/ui_state.py`.** Per-screen state objects. On any update, builds the relevant snapshot and pushes via the bridge. Rate-limit 5 msg/s per screen (per `docs/spec/02-serial-protocol.md`).
    Implemented in **Task A3**.

11. **New protocol message types** (pydantic models on daemon, C structs on firmware):
    - Host → ESP: `top_bar`, `screen_1`, `screen_2`, `screen_3`, `screen_4`, `pomodoro_fullscreen`
    - ESP → Host: `task_clicked`, `pr_clicked`, `notification_clicked`, `todo_clicked`, `macro_trigger`, `pomodoro_action`
    Field shapes match `docs/spec/02-serial-protocol.md` (lines ~370–471 catalog the relevant types).
    Implemented in **Task A2** (daemon) and **Task B4** (firmware).

12. **Macro button on top bar → opens a modal list of macros from `config.yaml`.**
    For M2 the modal is just a stub overlay showing "macros TBD" — full implementation in M6.
    Implemented in **Task B6**.

13. **Time / weather / claude_usage / pomodoro_counter source → all from the host's `top_bar` message.**
    Firmware never computes these. The clock the firmware displays is whatever the daemon's mock currently pushes.

14. **Pomodoro fullscreen overlay → separate LVGL widget above the carousel, hidden by default, shown when `pomodoro_fullscreen.visible=true` arrives.**
    For M2: just renders the elapsed-time text and a "stop" button (the button sends `pomodoro_action.action=stop`).

```python
"""In-memory aggregator that builds per-screen snapshots and pushes them to the bridge.

Per spec/02-serial-protocol.md: host pushes full snapshots, rate-limit 5 msg/s/screen.
"""

import asyncio
import time
from typing import TYPE_CHECKING

import structlog

from deskstation.bridge.protocol import (
    JiraTask, MeetingBar, Notification, PomodoroFullscreenData,
    PomodoroFullscreenMsg, PullRequest, Screen1Data, Screen1Msg,
    Screen2Data, Screen2Msg, Screen3Data, Screen3Msg, Screen4Data, Screen4Msg,
    StandupItem, TodoItem, TopBarData, TopBarMsg,
)

if TYPE_CHECKING:
    from deskstation.bridge.interface import BridgeProtocol

log = structlog.get_logger(__name__)

_RATE_LIMIT_PER_SEC = 5
_MIN_INTERVAL_SEC = 1.0 / _RATE_LIMIT_PER_SEC


class UIState:
    """Per-screen mutable state. Setters trigger a dispatch (rate-limited)."""

    def __init__(self, bridge: "BridgeProtocol") -> None:
        self._bridge = bridge
        self._last_send: dict[str, float] = {}
        self._pending: dict[str, asyncio.Task[None]] = {}

        # Screen state
        self._top_bar = TopBarData(
            clock="--:--", date="", weather="", claude_usage="",
            pomodoro_counter=0,
        )
        self._screen_1 = Screen1Data()
        self._screen_2 = Screen2Data()
        self._screen_3 = Screen3Data()
        self._screen_4 = Screen4Data()
        self._pomodoro_overlay = PomodoroFullscreenData(visible=False)

    # ---- setters ----

    def set_top_bar(self, data: TopBarData) -> None:
        self._top_bar = data
        self._schedule_send("top_bar")

    def set_screen_1(self,
                     today_tasks: list[JiraTask] | None = None,
                     queued_tasks: list[JiraTask] | None = None,
                     next_meeting: MeetingBar | None = None) -> None:
        if today_tasks is not None: self._screen_1.today_tasks = today_tasks
        if queued_tasks is not None: self._screen_1.queued_tasks = queued_tasks
        self._screen_1.next_meeting = next_meeting
        self._schedule_send("screen_1")

    def set_screen_2(self, notifications: list[Notification]) -> None:
        self._screen_2.notifications = notifications
        self._schedule_send("screen_2")

    def set_screen_3(self,
                     prs: list[PullRequest] | None = None,
                     standup: list[StandupItem] | None = None) -> None:
        if prs is not None: self._screen_3.prs = prs
        if standup is not None: self._screen_3.standup = standup
        self._schedule_send("screen_3")

    def set_screen_4(self, items: list[TodoItem]) -> None:
        self._screen_4.items = items
        self._schedule_send("screen_4")

    def set_pomodoro_overlay(self, data: PomodoroFullscreenData) -> None:
        self._pomodoro_overlay = data
        self._schedule_send("pomodoro_fullscreen")

    # ---- dispatch ----

    def _build_msg(self, key: str) -> object:
        if key == "top_bar": return TopBarMsg(data=self._top_bar)
        if key == "screen_1": return Screen1Msg(data=self._screen_1)
        if key == "screen_2": return Screen2Msg(data=self._screen_2)
        if key == "screen_3": return Screen3Msg(data=self._screen_3)
        if key == "screen_4": return Screen4Msg(data=self._screen_4)
        if key == "pomodoro_fullscreen":
            return PomodoroFullscreenMsg(data=self._pomodoro_overlay)
        raise ValueError(f"unknown key: {key}")

    def _schedule_send(self, key: str) -> None:
        """Rate-limited dispatch. If a send is already pending for this key, drop."""
        if key in self._pending and not self._pending[key].done():
            return  # coalesce — pending send will pick up the latest state
        self._pending[key] = asyncio.create_task(self._send(key))

    async def _send(self, key: str) -> None:
        elapsed = time.monotonic() - self._last_send.get(key, 0.0)
        if elapsed < _MIN_INTERVAL_SEC:
            await asyncio.sleep(_MIN_INTERVAL_SEC - elapsed)
        try:
            msg = self._build_msg(key)
            await self._bridge.send(msg)  # type: ignore[arg-type]
            self._last_send[key] = time.monotonic()
            log.debug("ui_state_sent", screen=key)
        except Exception as e:
            log.warning("ui_state_send_failed", screen=key, error=str(e))

    async def resend_all(self) -> None:
        """Push current state for every screen + top_bar. Use on reconnect."""
        for key in ("top_bar", "screen_1", "screen_2", "screen_3", "screen_4",
                    "pomodoro_fullscreen"):
            try:
                await self._bridge.send(self._build_msg(key))  # type: ignore[arg-type]
                self._last_send[key] = time.monotonic()
            except Exception as e:
                log.warning("ui_state_resend_failed", screen=key, error=str(e))
```

**Files:**

The current setup uses 800×60 partial buffers + `esp_lcd_panel_draw_bitmap`. Switch to **direct mode**: hand LVGL the panel's framebuffer pointers, set `disp_drv.direct_mode = 1`, and make `flush_cb` a no-op (or a `lv_disp_flush_ready` only, since the panel scans the buffer continuously).

- [ ] **Step 1: Rewrite `ui_init`.**

```c
esp_err_t ui_init(esp_lcd_panel_handle_t panel, esp_lcd_touch_handle_t touch)
{
    s_panel = panel;
    s_touch = touch;

    lv_init();

    // Direct mode: hand LVGL the panel's own framebuffers. flush_cb is a no-op
    // because the panel continuously scans the framebuffer — no copy needed.
    void *fb0 = NULL;
    void *fb1 = NULL;
    ESP_RETURN_ON_ERROR(
        esp_lcd_rgb_panel_get_frame_buffer(panel, 2, &fb0, &fb1),
        TAG, "get fb");

    lv_disp_draw_buf_init(&s_draw_buf, fb0, fb1, 800 * 480);

    lv_disp_drv_init(&s_disp_drv);
    s_disp_drv.hor_res = 800;
    s_disp_drv.ver_res = 480;
    s_disp_drv.flush_cb = flush_cb;
    s_disp_drv.draw_buf = &s_draw_buf;
    s_disp_drv.direct_mode = 1;
    s_disp_drv.full_refresh = 1;
    lv_disp_drv_register(&s_disp_drv);

    // ... rest of indev + lvgl_task setup unchanged ...
}
```
